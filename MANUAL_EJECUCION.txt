INSTRUCCIONES PARA EJECUTAR D&D WARGAMES MANUALMENTE
======================================================

Si el script automático no funciona, sigue estos pasos:

1. ABRIR CMD COMO ADMINISTRADOR:
   - Presiona Win + X
   - Selecciona "Windows PowerShell (Administrador)" o "Símbolo del sistema (Administrador)"

2. EJECUTAR COMANDOS UNO POR UNO:

   PRIMERO: Configurar Java (si no esta configurado)
   set JAVA_HOME=C:\Ruta\A\Tu\JDK
   set PATH=%JAVA_HOME%\bin;%PATH%

   SEGUNDO: Ir al directorio del proyecto
   cd /d %~dp0

   TERCERO: Compilar (si es necesario)
   if not exist bin mkdir bin
   dir /s /b src\com\dnd\wargames\*.java > .sources.list
   "%JAVA_HOME%\bin\javac.exe" -d bin @.sources.list
   del .sources.list

   CUARTO: Ejecutar tests
   "%JAVA_HOME%\bin\java.exe" -cp bin com.dnd.wargames.test.SimpleTest
   "%JAVA_HOME%\bin\java.exe" -cp bin com.dnd.wargames.test.BasicTest
   "%JAVA_HOME%\bin\java.exe" -cp bin com.dnd.wargames.test.MoraleAndWebSmokeTest

   QUINTO: Ejecutar la demo
   "%JAVA_HOME%\bin\java.exe" -cp bin com.dnd.wargames.demo.CombatDemo

   SEXTO: Ejecutar CLI interactivo
   "%JAVA_HOME%\bin\java.exe" -cp bin com.dnd.wargames.DndWargames

PROBLEMAS COMUNES:
==================

1. "java no reconocido":
   - Verifica que JDK esté instalado
   - Ejecuta como Administrador

2. "No se encuentra el archivo":
   - Verifica que estés en el directorio correcto
   - Ruta: %~dp0

3. Errores de compilación:
   - Los archivos ya deberían estar compilados
   - Si no, ejecuta el comando de compilacion

INSTRUCCIONES PARA EJECUTAR D&D WARGAMES MANUALMENTE
======================================================

Si el script automático no funciona, sigue estos pasos:

1. ABRIR CMD COMO ADMINISTRADOR:
   - Presiona Win + X
   - Selecciona "Windows PowerShell (Administrador)" o "Símbolo del sistema (Administrador)"

2. EJECUTAR COMANDOS UNO POR UNO:

   PRIMERO: Configurar Java
   set JAVA_HOME=C:\Program Files\Eclipse Adoptium\jdk-25.0.2.10-hotspot
   set PATH=%JAVA_HOME%\bin;%PATH%

   SEGUNDO: Ir al directorio
   cd C:\OneDrivePersonal\OneDrive\VS_Workspace\dnd_wargames

   TERCERO: Compilar (si es necesario)
   "%JAVA_HOME%\bin\javac.exe" -cp src src\com\dnd\wargames\demo\CombatDemo.java

   CUARTO: Ejecutar tests
   "%JAVA_HOME%\bin\java.exe" -cp src com.dnd.wargames.test.SimpleTest
   "%JAVA_HOME%\bin\java.exe" -cp src com.dnd.wargames.test.BasicTest

   QUINTO: Ejecutar la demo
   "%JAVA_HOME%\bin\java.exe" -cp src com.dnd.wargames.demo.CombatDemo

   SEXTO: Ejecutar CLI interactivo
   "%JAVA_HOME%\bin\java.exe" -cp src com.dnd.wargames.cli.CommandLineInterface

==================

2. "No se encuentra el archivo":
   - Verifica que estés en el directorio correcto
